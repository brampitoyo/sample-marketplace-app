FAQ for marketplace.config

Some questions you might have about this file and why we're working on it.

Is this the only way I can submit an app?

No, this is one of the ways to submit an app. We think it will be most useful to developers who prefer to work with files and/or at the command line.

How do you declare a free app with in-app payment?

Set the value of the `price` field to `free`, and declare at least one product under `in_app_products`. The system will set the price tier to "tier 0" with in-app payments
